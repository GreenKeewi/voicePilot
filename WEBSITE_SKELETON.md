## NAVIGATION BAR (Dock Style)

### Layout Structure
```
[LOGO: VoicePilot]  |  Home  |  Products & Services  |  About  |  Contact  |  Endorsements  |  Blog
```

### Navigation Design Specifications
- **Position**: Horizontal dock-style navigation bar at top of page
- **Style**: Fixed/sticky header that remains visible when scrolling
- **Background**: Semi-transparent white (#FFFFFF with 95% opacity)
- **Border**: Subtle bottom border using accent color (#3B82F6, 1px solid)
- **Link Styling**: 
  - Default: #1E293B (Dark Slate)
  - Hover: #3B82F6 (Primary Blue)
  - Active Page: #3B82F6 with underline
- **Logo Position**: Left-aligned
- **Nav Links**: Centered or right-aligned
- **Mobile**: Hamburger menu for responsive design

## COLOUR PALETTE + FONTS

### Primary Colour Palette
1. **Primary Blue**: `#3B82F6` 
   - Use for: CTAs, links, accents, brand highlights
   
2. **Deep Navy**: `#1E293B`
   - Use for: Headers, primary text, professional elements
   
3. **Light Slate**: `#64748B`
   - Use for: Secondary text, descriptions, subtitles
   
4. **Sky Blue**: `#E0F2FE`
   - Use for: Background sections, cards, highlights
   
5. **Pure White**: `#FFFFFF`
   - Use for: Main background, cards, contrast areas
   
6. **Accent Teal**: `#06B6D4`
   - Use for: Icons, secondary CTAs, decorative elements

### Typography
**Primary Font**: **Inter** or **Poppins** (Google Fonts)
- Use for: All body text, descriptions
- Weights: 400 (Regular), 500 (Medium), 600 (Semi-Bold)

**Heading Font**: **Raleway** or **Montserrat** (Google Fonts)
- Use for: H1, H2, H3 headings
- Weights: 600 (Semi-Bold), 700 (Bold), 800 (Extra-Bold)

### Font Size Guidelines
- H1 (Hero): 48-56px
- H2 (Section Headers): 36-42px
- H3 (Subsections): 24-28px
- Body Text: 16-18px
- Small Text/Captions: 14px

## IMPLEMENTATION NOTES FOR WIX/GOOGLE SITES/CANVA

### Page-by-Page Setup Tips

#### For Wix:
1. Use "Blank Template" and build with sections
2. Add "Strip" backgrounds with Sky Blue (#E0F2FE) for alternating sections
3. Use Wix's built-in contact form widget (styled to match colors)
4. Embed Google Maps using HTML iframe embed element
5. Add social media bar from Wix's social tools
6. Use Wix Blog template for blog section

#### For Google Sites:
1. Create pages using left sidebar page manager
2. Use "Layouts" for column structures (2-column for product comparisons)
3. Add images via "Insert ‚Üí Image" and use image carousels
4. Embed Google Maps directly (Sites has native integration)
5. Create contact form using Google Forms, embed on Contact page
6. Use "Button" elements styled with hex color #3B82F6

#### For Canva:
1. Start with "Website" template category
2. Create 6 separate pages as per site map
3. Use "Brand Kit" feature to save color palette
4. Design each page with consistent typography (Poppins/Inter from Canva fonts)
5. Export as separate page images or use Canva's website publishing
6. Add links between pages using Canva's hyperlink feature
